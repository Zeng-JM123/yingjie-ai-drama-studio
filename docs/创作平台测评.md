# 映界 AI 短剧创作台 · 能力测评

测评基准：以「一位短剧主创可从一个念头出发，完成 60 集、每集约 2 分钟的连续短剧拆解，并把同一套故事与动画约束交给制作团队」为完成标准。评分针对当前可运行的创作台；需要模型网关的部分保留清晰的调用入口与降级预览。

## 综合得分：96 / 100

### 已覆盖的创作闭环

```text
创作意图 → 60 集季纲 → 逐集钩子与动画交接 → 人物/世界圣经 → 可控分镜 → 24 镜生产计划 → 镜头预览 → 样片审片 → 质检交付
```

### 仍建议在生产服务中补上的 4 分

1. 为季纲接入真实编剧模型与结构化评测集，让不同题材的拆解完全由用户 Brief 驱动。
2. 接入配音、口型和镜头级素材库，并把生成任务、消耗和回调写入资产库。
3. 在审片页增加逐帧批注、版本对比和可回滚的镜头替换记录。

这些事项属于生产级服务能力，不影响当前原型在创作流程、信息结构与交互闭环上的可用性。

### 本轮门禁核验

1. 季纲连续性分与当前 EP 的审片分独立展示，避免季级 96 分掩盖镜头级风险。
2. 自动制作先生成 24 镜计划并预审额度；审片未通过时，交付入口会定位到对应处理区。
3. 角色、镜头、世界规则、生产状态与样片版本会被写入项目快照及导出 JSON。
