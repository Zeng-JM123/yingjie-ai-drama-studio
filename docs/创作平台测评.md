# 映界 AI 短剧创作台 · 能力测评

测评基准：以「一位短剧主创可从故事意图推进到可审样片，并可把同一套创作约束交给制作团队」为完成标准。评分针对当前可运行的前端创作台；需要模型网关的部分保留清晰的调用入口与降级预览。

| 核心维度 | 验收点 | 得分 |
| --- | --- | ---: |
| 故事意图与节奏 | 可编辑 Brief、题材与画幅约束、三幕情绪曲线、Agent 拆解反馈 | 14 / 15 |
| 角色与世界圣经 | 双角色三视图预览、服化选择、表演/声线/视觉锚点、可新增角色蓝图、世界规则 | 16 / 17 |
| AI 分镜设计 | 8 个可选镜头、景别/运镜/时长/情绪/台词编辑、镜头提示词、拆分/新增/重排 | 16 / 17 |
| 生成资产闭环 | 角色预览、镜头预览入口、Seedance 网关自动接入、未配网时的可演示降级状态 | 14 / 15 |
| 表演与声音 | 角色声线锚点、试听状态、台词与环境音字段、声音 Agent 回写 | 11 / 12 |
| 样片与质检 | 竖屏样片、带时间码播放、审片人入口、9 项质检清单、风险定位与低风险修复 | 13 / 14 |
| 交付与协作 | 项目 JSON 导出、审片包入口、交付门禁、可追溯角色/镜头参数 | 10 / 10 |

## 综合得分：94 / 100

### 已覆盖的创作闭环

```text
创作意图 → 剧本节奏 → 人物/世界圣经 → 可控分镜 → 镜头预览 → 样片审片 → 质检交付
```

### 仍建议在生产服务中补上的 6 分

1. 将角色、镜头、批注与版本历史持久化到项目服务，支持多人实时协作和权限控制。
2. 接入真实的图像、视频、配音与口型模型，并把生成任务、消耗和回调写入资产库。
3. 在审片页增加逐帧批注、版本对比和可回滚的镜头替换记录。

这些事项属于生产级服务能力，不影响当前原型在创作流程、信息结构与交互闭环上的可用性。
